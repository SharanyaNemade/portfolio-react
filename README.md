# 🌐 Personal Portfolio Website

This is my **React-based online portfolio** showcasing my skills, projects, resume, and contact details.  
The site is fully responsive, optimized for performance, and deployed for global access.  

---

## 🚀 Features
- **About Me** section with profile image and introduction  
- **Projects** showcase with descriptions and links  
- **Skills** section highlighting technical expertise  
- **Resume** download option (PDF)  
- **Contact** form with details  
- Responsive design for all devices  
- Optimized production build with lazy loading  

---

## 🛠️ Tech Stack
- **Frontend:** React, React Router, CSS  
- **Deployment:** Netlify / Vercel / GitHub Pages  
- **Version Control:** Git, GitHub  

---

## ⚙️ Installation & Setup

1. Clone the repository  
   ```bash
   git clone https://github.com/<your-username>/portfolio-react.git
   cd portfolio-react

2. Install dependencies

npm install


3. Run in development mode

npm start


4. Build for production

npm run build

📌 Challenges & Solutions

Image & Resume Assets

Issue: React build failed due to missing image/PDF paths.

Solution: Placed static assets in /public and referenced them correctly.

Responsive Design

Challenge: Ensuring layout looked good across devices.

Solution: Used CSS Flexbox/Grid + media queries.

Performance Optimization

Challenge: Slow load times due to large files.

Solution: Implemented lazy loading, minified JS/CSS, and optimized images.
